> ## 开发模式

### 参数配置列表

```
{
	'type': 'TheButton',
	'normal': {
		'backgroundColor': '#31b16c',
		'fontSize': '20',
		'borderRadius': 5,
		'borderColor': '#31b16c',
		'borderWidth': 1,
		'borderStyle': 'solid',
		'color': '#fff',
		'lineHeight': '47px',
		'textAlign': 'center',
		'cursor': 'pointer',
		'opacity': 0.6
	},
	'mouse': {
		'backgroundColor': '#31b16c',
		'borderRadius': 5,
		'fontSize': '20',
		'borderColor': '#31b16c',
		'borderWidth': 1,
		'borderStyle': 'solid',
		'color': '#fff',
		'lineHeight': '47px',
		'textAlign': 'center',
		'cursor': 'pointer',
		'opacity': 0.8
	},
	'emphasis': {
		'backgroundColor': '#31b16c',
		'borderRadius': 5,
		'fontSize': '20',
		'lineHeight': '47px',
		'color': '#fff',
		'textAlign': 'center',
		'borderColor': '#31b16c',
		'borderWidth': 1,
		'borderStyle': 'solid',
		'opacity': 0.7
	},
	'selected': false,
	'text': '确 认'
}
```
